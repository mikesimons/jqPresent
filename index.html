What?

jqPresent is a jQuery plugin that allows you to give presentations written in html.

Why?

Because I could.

But also:

  • I don't have ready access to a mac with keynote
  • I don't own a copy of powerpoint
  • Open office generally crashes a million times a day
  • I can write HTML / CSS faster than I can mess about with a GUI
  • I can write my own sexy transitions

How do I use it?

Dead simple

  • Signup to Github
  • Fork the jqPresent repo
  • Clone your fork
  • Hack away at this presentation

What can it do?

  • Per slide transitions
  • Points with per point transitions
  • All styled by CSS
  • Can embed any flash movies / iframes as normal or load them in prettyphoto etc
  • Optional pager and relative slide navigation
  • Events per slide / point enter and exit for custom logic
